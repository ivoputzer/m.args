**[m(icro)](https://github.com/ivoputzer/m.cro#readme)[m.args](https://github.com/ivoputzer/m.args)** is a lightweight arguments parser written in es6+

# Usage
```js
  #/usr/bin/env node
  const {parse} = require('m.args')
  const args = parse(['--port', '8080']) // { port: 8080 }
```
